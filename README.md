## A simple, Sony Playstation 1 (PSX) emulator written in C23


## Features:
* Built using [C23](https://en.cppreference.com/w/c/23) features
* Runs on amd64 (x86_64 Intel/AMD PC)
* Upstream compiler support (CLANG-14 & GCC-12)
